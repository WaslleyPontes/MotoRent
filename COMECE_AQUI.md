# 🚀 COMEÇAR AGORA - Quick Start

Seu projeto está **100% pronto**. Escolha uma das 3 formas abaixo:

---

## 🎯 FORMA 1: Script Automático (RECOMENDADO)

**Copie e cole uma destas linhas:**

### PowerShell
```powershell
.\run.ps1
```

### CMD (Windows)
```cmd
run.bat
```

Pronto! O script vai:
1. ✅ Criar ambiente virtual (se não existir)
2. ✅ Ativar ambiente virtual
3. ✅ Instalar dependências (se não estiverem)
4. ✅ Criar banco de dados (se não existir)
5. ✅ Iniciar servidor na porta 5000

---

## 🎯 FORMA 2: Manual (3 comandos)

```powershell
# 1. Ativar ambiente
venv\Scripts\Activate

# 2. Instalar dependências (primeira vez)
pip install -r requirements.txt

# 3. Rodar
python app.py
```

---

## 🎯 FORMA 3: Uma linha (depois da primeira vez)

```powershell
venv\Scripts\Activate; python app.py
```

---

## 🌐 ACESSAR A APLICAÇÃO

Após qualquer um dos 3 métodos acima, abra seu navegador e vá para:

```
http://127.0.0.1:5000
```
