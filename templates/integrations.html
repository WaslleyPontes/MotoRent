Integração com Mapas

O MotoRent oferece suporte a múltiplas plataformas de mapeamento para máxima flexibilidade.

🗺️ Google Maps

Para ativar o Google Maps em tempo real na telemetria:

  1. Acesse Google Cloud Console
  2. Crie um novo projeto
  3. Ative as APIs: "Maps JavaScript API" e "Directions API"
  4. Gere uma chave de API com restrições geográficas
  5. Adicione a chave no arquivo templates/telemetry.html:
<script src="https://maps.googleapis.com/maps/api/js?key=SUA_API_KEY></script>

Custo: Primeiros 28.500 carregamentos/mês são gratuitos

Link: Google Maps Documentation

🗺️ OpenStreetMap (Leaflet)

Já integrado! Sem custos, sem restrições. Usado como fallback quando Google Maps não está disponível.

Link: OpenStreetMap

📱 Waze

Integre Waze para navegação integrada dos motoristas:

  1. Adicione links de navegação Waze nas informações de destino
  2. Use a URI: https://waze.com/ul?ll=LAT,LNG&navigate=yes
  3. Exemplo em HTML:
<a href="https://waze.com/ul?ll=-3.73,-38.54&navigate=yes" target="_blank"> Abrir no Waze </a>

Vantagens: Navegação em tempo real, alertas de trânsito, comunidade de motoristas

Link: Waze Developer

📍 Apple Maps

Para usuários iOS:

maps://maps.apple.com/?ll=LAT,LNG&q=Search%20Term
